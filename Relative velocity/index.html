What is actually happenning!

In this relative velocity project, we have used two cases. one is the movement of a single object and the other is movements of two separate objects. In here, the movement for a single object is identified as "Object 01" and movements for two objects as "Object 02"



for object 1 simulation


This is the interface for the object one.




  • By clicking the "SWITCH" button which is numbered as No:1 you can be shifted easily into object 2.


  • By clicking the "Insert Data" button which is numbered as No:2 is used for inserting data


  • By clicking the "Instruction" button which is numbered as No:1 is used for usage guidance.


for object 2 simulation


This is the interface for Object two.

  • By clicking the "SWITCH" button which is numbered as No:1 you can be shifted easily into object 1.


  • By clicking the "Insert Data" button which is numbered as No:2 , is used for inserting data


  • By clicking the "Instruction" button which is numbered as No:1 is used for usage guidance.








  • By clicking the "Insert Data" button we can get this fill form to insert data.


  • We can insert the degree by either filling the No.1 blank "Counter clockwise degree from the river edge" or by simply adjusting the button "A" with the mouse. click confirm after inserting all the data respectively.







  • After confirming the data form, to see the object movement, click "START". The object movement and the answers for all the calculations will be displayed afterwards.







  • As in object 1 fill the data for object 2








  • After filling and confirming the data respectively, the movement of the two objects can be seen.


  • For that clicking the "START" button is necessary.


  • After clicking the "START" button you can see the answers for the movements of the both objects, whether they are supposed to meet each other or not, and if they are supposed to meet each other what will be the distance and how long that it will be taken to .The answers for all the calculations will be displayed afterwards.
